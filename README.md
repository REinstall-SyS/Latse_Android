# Latse (Android) 🏔️

Latse is a dedicated Android application designed to ensure the safety, readiness, and navigation of trekkers tackling high-altitude trails. Built with offline capabilities in mind, it serves as a digital Sherpa for routes like Everest Base Camp, Annapurna Base Camp, and the Langtang Valley.

## ✨ Key Features

* **Live Environmental Dashboard:** Real-time tracking of GPS altitude, temperature, wind speed, and effective atmospheric oxygen levels.
* **Offline Trail Mapping:** Interactive maps with critical waypoints (start points, towns, medical aid posts, and high-risk zones) for major trekking routes.
* **Altitude Health Check (LLS):** A built-in clinical assessment tool utilizing the international standard Lake Louise Score to check for Acute Mountain Sickness (AMS) and red-flag symptoms.
* **Sherpa Toolkit:** * Offline English-to-Nepali emergency phrase translator with phonetic pronunciations.
  * SOS Strobe & Hardware Flashlight.
  * Native hardware compass.
  * Naismith's Rule pace calculator for estimating remaining hike times.
* **Emergency Protocols:** Direct access to official Nepal emergency contacts (Tourist Police, Ambulance) and helicopter rescue guidelines.
* **Readiness Audit:** Pre-trek physical and gear assessment checklists.
